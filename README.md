SIMPLE PASSWORD GENERATOR

A beginner-friendly command-line Password Generator built using Python. This application allows users to create random passwords by selecting the desired password length and choosing whether to include uppercase letters, numbers, and special symbols.
The project demonstrates fundamental Python programming concepts while providing a practical tool for generating secure and customizable passwords.

FEATURES
-> Generate random passwords of any desired length
-> Option to include uppercase letters
-> Option to include numbers
-> Option to include special symbols
-> User-friendly command-line interface
-> Input validation for password length and user choices
-> Randomized password generation using Python's built-in `random` module

HOW IT WORKS
The program first asks the user to specify the desired password length. It then allows the user to choose whether uppercase letters, digits, and symbols should be included in the password. Based on these preferences, the application creates a pool of characters and randomly selects characters to generate a password of the specified length.

TECHNOLOGIES USED
-> Python 3
-> Functions
-> Loops
-> Conditional Statements (`if/else`)
-> String Manipulation
-> Random Module (`random.choice()`)

GETTING STARTED

1. Clone the repository.
2. Ensure Python 3 is installed on your system.
3. Run the program:

python password_generator.py

4. Follow the prompts to customize and generate your password.
